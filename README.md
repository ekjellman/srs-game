# srs-game

An RPG you play with only the arrow keys! :arrow_backward: :arrow_up_small: :arrow_down_small: :arrow_forward:

## Quick Start

### Set up Conda environment
```
conda env create -f environment.yml
source activate srs-game
```

### Run tests
```
bin/unit-tests
```

### Play the game

#### Desktop version
```
bin/play-game
```

#### Browser version
```
bin/build-html
open srs.html
```
